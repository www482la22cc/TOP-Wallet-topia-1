## Environment Setup
Install [Node.js 16](https://nodejs.org) and npm

## Running Locally

```bash
npm install --legacy-peer-deps
npm run start
```
Open your browser and visit http://127.0.0.1:3000

## Production Build
```bash
npm run build
```

Build the project to the ./dist/ folder with npm run build.
